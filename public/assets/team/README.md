# Team Photo
Add team member photos here

## File Naming
Name files like: `firstname-lastname.jpg`

Examples:
- `ethan-guo.jpg`
- `jake-rosenblum.jpg`
- `member1.jpg`

## Image Specs
- **Size**: 200x200px minimum
- **Format**: JPG, PNG, or WebP
- **Style**: Square photos work best (cropped to circle)
- **File size**: Keep under 500KB each

## How It Works
The website will automatically load photos from this folder. If a photo is missing, it shows initials instead!
